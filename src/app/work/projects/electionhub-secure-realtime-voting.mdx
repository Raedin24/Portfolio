# ElectionHub – Secure Real-Time Voting

End-to-end digital voting platform delivering **instant results**, **OTP-based authentication**, and **verifiable audit trails** for transparent online elections.

---

## Overview

ElectionHub enables electoral bodies to manage multiple concurrent elections with strong identity verification, real-time dashboards, and tamper-evident logging.  
Built with a focus on **security, transparency, and scalability**, it has processed over **4,000 verified votes** with zero downtime.

---

## Role & Responsibilities

- Architected backend (Django REST Framework + PostgreSQL)  
- Implemented OTP login, device fingerprinting, and JWT session handling  
- Built real-time tally updates using Django Channels and WebSockets  
- Designed hash-chained audit logs for provable integrity  
- Deployed via Docker and Nginx on hardened DigitalOcean infrastructure  

---

## Architecture

- **Frontend:** React + Vite + Tailwind CSS + Recharts (for live dashboards)  
- **Backend:** Django REST Framework with Redis caching and rate limiting  
- **Database:** PostgreSQL (ACID-compliant, normalized schema)  
- **Realtime:** Django Channels / WebSockets for vote streams  
- **Security:** OTP auth, JWT tokens, hash-linked logs, and strict RBAC  
- **Infra:** Dockerized services behind Nginx + HTTPS (Let’s Encrypt)  

---

## Key Features

- Multi-election and multi-role support (Voter, Admin, EC)  
- Live dashboards with real-time vote counts  
- Device-bound OTP login and replay protection  
- Append-only audit logs for verifiable transparency  
- Automated deploys and encrypted backups  

---

## Outcome

Adopted by student election commissions for transparent digital voting, achieving real-time results, secure authentication, and provable system integrity.
