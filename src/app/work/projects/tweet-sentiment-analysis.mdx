## Overview

This project implements a comprehensive **tweet sentiment analysis** system using machine learning techniques. The model classifies tweets into three sentiment categories: **positive**, **negative**, and **neutral**, leveraging advanced NLP preprocessing and multiple classification algorithms.

---

## Key Features

- Advanced text preprocessing pipeline with tokenization, stemming, and lemmatization
- TF-IDF vectorization for feature extraction
- Multiple classifier implementations (SVM and Logistic Regression)
- Comprehensive model evaluation with accuracy, precision, recall, and F1-score
- Real-time sentiment prediction capabilities
- Data visualization for class distribution analysis

---

## Technical Approach

The project follows a systematic machine learning workflow:

**Data Preprocessing**: Implements a robust text cleaning pipeline including:
- URL and HTML tag removal
- Tokenization and punctuation removal
- Stopword elimination using NLTK
- Stemming with Snowball Stemmer
- Lemmatization with WordNet

**Feature Engineering**: Utilizes TF-IDF vectorization with optimized parameters to convert text into numerical features while handling the high dimensionality of tweet data.

**Model Selection**: Implements and compares multiple classifiers:
- Support Vector Machine (SVM) with linear kernel
- Logistic Regression with optimized iterations

---

## Implementation Details

The sentiment analysis pipeline consists of several key components:

1. **Data Loading & Exploration**: Analysis of class distribution and data quality
2. **Text Cleaning**: Comprehensive preprocessing to handle Twitter-specific text patterns
3. **Feature Extraction**: TF-IDF vectorization with max_features=2500 for dimensionality control
4. **Model Training**: 80-20 train-validation split with proper cross-validation
5. **Evaluation**: Multi-class classification metrics and detailed performance analysis

---

## Results

The models achieved strong performance on the tweet sentiment classification task:

**Logistic Regression Results:**
- Accuracy: 68%
- Precision: 0.70 (macro average)
- Recall: 0.67 (macro average)
- F1-score: 0.68 (macro average)

**SVM Results:**
- Accuracy: 68%
- Precision: 0.72 (macro average)
- Recall: 0.65 (macro average)
- F1-score: 0.68 (macro average)

The system shows particularly strong performance in identifying positive sentiments while effectively handling the challenges of neutral sentiment classification.
